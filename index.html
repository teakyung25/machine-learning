What I Learned

Cost Function

The following is the cost function, it finds the mean error between the point and the hypothesis. The hypothesis is the line all examples in the dataset will be compared to.

How it Works?

The hypothesis is represented as h(x). The theta 0 and theta 1 determines the start and slope of the line. The hypothesis is subtracted by the real y value of the data point and squared, then this process repeats m(the number of data examples) number of times(represented by the sigma) while being added together, then the sum of the differences are divided by 2*m to get the mean.

Gradient Descent

Since, the cost function can only find the mean difference between one hypothesis and the data examples, another function is required to compare and find the local or global optima(the best hypothesis for the data examples). This function is called the Gradient Descent Algorithm.

How it Works?

The function will either add or subtract to theta j each iteration. This is because the derivative is the slope of the line tangent to the point (theta j, J(theta j)), so if the slope is negative the function will add to theta j and if the slope is positive, the function will subtract to theta j. This process repeats until the point is at the local minimum. The alpha in the equation is the learning rate, it represents how big of a step will be taken, but the alpha is affected by the derivative, so even if the alpha is kept constant, the alpha will decrease as the slope of the derivative decreases with it.
